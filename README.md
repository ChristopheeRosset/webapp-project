These instructions are meant for a repository that will be clone within a linux distro or wsl
1. Clone the repository
2. Create virtual environment at the repository source: python -m venv .venv
3. Activate it: <source .venv/bin/activate>
4. Install dependencies: pip install -r server/requirements.txt
5. Within config/, create <config.dci.yaml> and complete global var as you prefer
6. Launch the server with <python3 -m server.app>
